# Supplychain
# 汽配供应链顾客端SpringBoot后端
> 此仅为后台  
>### [vue前台点这里](https://github.com/feihb123/vue-cli-demo)

技术栈：
- Springboot
- MyBatis
- JPA
- Shiro
- AliPay
- Redis

>（需要安装Lombok启动）

### 此项目基于[SupplyChain](https://github.com/feihb123/supplychain)开发
- 原项目前后端采用:Springboot+Thymeleaf
- 现版本使用：Springboot+Vue
- 本项目仅保留顾客角色，且实现前后端分离
